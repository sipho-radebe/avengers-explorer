# Avengers Explorer

A static multi-page web application that allows users to explore Avengers characters, view their biographies, and anlayze their power stats.

## Features
- Homepage displaying Avengers characters
- Individual hero pages with:
  - Biography details
  - Power statistics
- Navigation bar with Home and Contact pages
- Clean and structured layout using HTML and CSS

## Tech Stack
- HTML5
- CSS

## Project Structure
- 'index.html' - homepage
- '*.html' - individual hero pages
- 'style.css' - styling
- 'images/' - hero images

## How to run
1. Clone the repository
2. Open 'index.html' in your browser
